URL Shortner Microservice

Overview:

This is a FreeCodeCamp API Project where a user can pass a URL as a parameter and will receive a shortened URL in the JSON response

Example Creation Usage:

https://ms-url-shortner.herokuapp.com/
https://ms-url-shortner.herokuapp.com/new/https://www.google.com

Example Output:

{ "original_url":"https://www.google.com", "short_url":"https://ms-url-shortner.herokuapp.com/xxxxxx" }

Usage:

https://ms-url-shortner.herokuapp.com/2523" }
will redirect to
https://www.google.com
